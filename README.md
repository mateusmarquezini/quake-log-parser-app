# Quake Log Parser - Solution

## 📝 Requirements 

- [Go 1.21.3](https://go.dev/doc/install)


## ▶️ Running the Application

#### Application

- Using Make

 ```bash
make run-app
```
- Using native Go command:

 ```bash
go run ./app
```

#### Running unit tests

- Using Make

 ```bash
make run-test
```
- Using native Go command:

 ```bash
go test -v ./...
```
